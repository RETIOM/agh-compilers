## Tabela tokenów

| KOD | Opis | Przykład |
| :--- | :--- | :--- |
| IDENTIFIER | Nazwa zmiennej lub funkcji (`[a-zA-Z][a-zA-Z0-9]*`) | `x`, `var1`, `speed` |
| NUMBER | Ciąg cyfr (`[0-9]+`) | `123`, `7`, `500` |
| PLUS | Operator dodawania (`+`) | `+` |
| MINUS | Operator odejmowania (`-`) | `-` |
| MULTIPLY | Operator mnożenia (`*`) | `*` |
| DIVIDE | Operator dzielenia (`/`) | `/` |
| LPAREN | Lewy nawias `(` | `(` |
| RPAREN | Prawy nawias `)` | `)` |
